How to Use

  1. Adjust the password length using the slider (8-64 characters)
  2. Select the character types you want to include in your password
  3. Click the "Generate" button to create a new password
  4. Use the refresh button next to the password to generate a new one
  5. Use the "Copy" button to copy the password to your clipboard
  6. The password strength indicator shows how secure your generated password is
